Goal

Refine Amy after the 1.0.6 release using the new screenshots: make Today feel like a real notes app instead of an overlaid text/rail hack, keep the floating top controls from clipping note text, improve keyboard clearance, simplify and beautify the food photo agent sheet, make stats show average-vs-goal instead of week-vs-goal, improve Goals/Targets and Local Data in Settings, make the Today widget height-resizable and quick-log focused, then build and publish a new local native release.

Current Findings

  • TodayScreen.tsx still uses one multiline TextInput with a separate absolute calorie rail. That is why note text can clip under the top controls while calories remain visible.
  • The top logo/date/streak/settings controls live in normal layout, but screenshots show the scroll/edit surface moving underneath them without enough guarded inset. They should behave as a floating header over padded content.
  • The food capture route uses ModalShell title Food agent and then repeats Food agent inside CaptureModal.
  • Stats cards currently show Week total / Goal, which compares seven-day totals to one-day goals. The header should show daily average against the goal.
  • Settings shows Goals/Targets and Local Data as raw form sections. They need clearer goal type/status, explicit macro targets, and cleaner export/import affordances.
  • The Today widget still displays note text and macro line. The requested direction is quick logging only: tap/type/camera/mic/saved/barcode, not "what I ate".

Design Direction

  • Register: product UI. The design should feel like a compact Android notes tool, not a dashboard or decorative health app.
  • Primary surface: logging. Top controls should hover as controls, while note rows are the content.
  • Use restrained dark product styling, existing tokens, lucide icons, large touch targets, and stable row heights. Avoid nested cards and oversized roundness.
  • Stats should compare like with like: daily average calories against daily calorie target, daily average protein/carbs/fat against daily macro targets.

  • Android emulator visual smoke of Today scroll/header/keyboard, Food agent, Stats, Settings Goals/Local Data, and widgets/deep links where available.
  • Native local Gradle release build, aapt2 dump badging, apksigner verify, SHA-256, GitHub release asset update.
