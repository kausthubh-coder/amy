# Amy 1.0.7

This release refines the daily notes surface, photo capture sheet, Stats, Goals, Local Data, and Android Today widget from the latest screenshot feedback.

## Fixes

- Today now keeps notes and calories inside one aligned scrolling log surface under floating header controls.
- The bottom dock has more keyboard clearance on Android.
- Food agent no longer repeats the same title inside the sheet, and its photo area is now one cleaner capture/preview panel.
- Label/photo capture accessibility copy now distinguishes meal photos from nutrition label photos.
- Stats now shows daily average against the goal instead of week total against a daily goal.
- Goals & Targets now shows goal type, status, macro targets, saved/pending values, and sanitized input saves.
- Local Data now has clearer private backup export/import controls and on-device data counts.
- The mini Today widget is quick-log focused and no longer displays what was eaten.
- The mini Today widget now includes vertical resize metadata and adaptive layout text.
