# Amy 1.0.6

This release fixes the logging, widget, stats, and streak issues reported from the Android screenshots.

## Fixes

- Photo and prompt captures now add a visible Today note line while logging calories.
- Today keeps the calorie rail aligned when long notes scroll.
- Streaks now use a current contiguous streak instead of total logged days.
- Streaks show today eaten, week total, and macro totals against goals.
- Stats now uses one top header with Stats/Streaks tabs and the close button.
- Macro circles now render real progress with distinct macro colors.
- Android widgets now use consistent fire calorie language, vector action icons, and improved resize metadata.
